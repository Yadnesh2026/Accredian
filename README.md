# Accredian – Enterprise Page Clone

## Project Overview

This project is a **partial clone of the Accredian Enterprise website**, built using modern web technologies.
The goal was to recreate the UI, structure, and responsiveness while maintaining clean and scalable code.

---

## Setup Instructions

Follow these steps to run the project locally:

### 1. Clone the repository

```bash
git clone https://github.com/Yadnesh2026/Accredian.git
cd Accredian
```

### 2. Install dependencies

```bash
npm install
```

### 3. Run the development server

```bash
npm run dev
```

### 4. Open in browser

```
http://localhost:3000
```

---

##  Approach Taken

* Built using **Next.js (App Router)** for better performance and scalability
* Designed a **component-based architecture** for reusability
* Focused on:

  * Clean UI structure
  * Responsive design (mobile + desktop)
  * Proper folder organization
* Used **modular CSS / styling** for maintainability
* Implemented **semantic HTML** for better accessibility
